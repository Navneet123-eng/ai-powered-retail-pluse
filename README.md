# RetailPulse AI Analysis

## Project Overview
This project analyzes Superstore sales and profit data to identify business trends and performance insights.

## Analysis Performed
- Sales by Category
- Sales by Region
- Profit by Category
- Profit by Sub-Category

## Tools Used
- Tableau
- CSV Dataset
- GitHub

## Key Insights
1. Technology category generates the highest sales.
2. East and West regions contribute the most revenue.
3. Tables and Bookcases are loss-making products.
4. Copiers and Phones are highly profitable.
